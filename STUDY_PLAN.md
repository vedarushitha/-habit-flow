### Structure Your Answers

1. **Start with Definition/Overview**
   - "React is a JavaScript library for building user interfaces..."

2. **Give Context/Example**
   - "In this project, I use React to create components like HabitCard..."

3. **Explain Why**
   - "I chose React because it provides component-based architecture..."

4. **Connect to Project**
   - "For example, when a user clicks 'Mark Done', React updates the state..."

   - "I'm not entirely sure about that, but based on my understanding..."
